# Configuration File

Dashboard Analytic reads runtime settings from environment variables.

Important variables:

- `APP_NAME`
- `APP_VERSION`
- `APP_PORT`
- `APP_DEV_PORT`
- `APP_ADMIN_USERNAME`
- `APP_ADMIN_PASSWORD`
- `APP_DATABASE_PATH`
- `APP_INPUT_DIR`
- `APP_OUTPUT_DIR`
- `APP_EXPORT_DIR`

For Docker usage, these values are defined in `docker/.env`.
